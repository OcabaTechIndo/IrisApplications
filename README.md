# Foobar

Foobar is a Python library for dealing with word pluralization.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
npm install
```

## Usage

```python
npm run build

# Server Dev
npm run start-dev

# returns 'geese'
npm run image-build

# Pilihan 
npm run start

```
